# API для управления мероприятиями

Студент: Качкаев Н.А.  
Группа: ПрИ-23

## Описание

REST API для управления мероприятиями на Node.js с использованием Express и PostgreSQL.

### Функциональность

- CRUD операции для мероприятий и пользователей
- Soft delete (мягкое удаление)
- Логирование запросов
- Swagger документация
- Ограничение на количество мероприятий, создаваемых пользователем в день

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Сервер будет запущен на http://localhost:5000

## API документация

После запуска сервера документация доступна по адресу http://localhost:5000/api-docs
