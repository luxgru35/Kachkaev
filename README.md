# Лабораторная работа №1: API для управления мероприятиями

**Студент:** Качкаев Н.А.  
**Группа:** ПрИ-23

## Содержание

- Backend: REST API на Node.js с Express и PostgreSQL

## Требования

- Node.js (v14+)
- npm
- PostgreSQL

## Установка и запуск

### Backend

```bash
cd backend
npm install
npm run dev
```

Сервер будет доступен по адресу: http://localhost:5000  
Документация API: http://localhost:5000/api-docs

## Реализованные требования

### Основные функции API
- ✓ POST /events - Создание мероприятия
- ✓ GET /events - Получение списка мероприятий
- ✓ GET /events/:id - Получение мероприятия по ID
- ✓ PUT /events/:id - Обновление мероприятия
- ✓ DELETE /events/:id - Удаление мероприятия
- ✓ POST /users - Создание пользователя
- ✓ GET /users - Получение списка пользователей
